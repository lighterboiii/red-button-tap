# Изображения лендинга

Структура:

- `hero/` — картинки для первого экрана (Hero).
- `cases/` — кейсы (например `skadi.jpg` для блока CaseSkadi).
- `team/` — фото команды (например `member-1.jpg`, `member-2.jpg`).

**Как подключать в компонентах (Vite):**

```js
import img from '../../assets/images/cases/skadi.jpg';
// в JSX: <img src={img} alt="..." />
```

Рекомендуется сжимать изображения (WebP/JPEG, ориентир 80–200 KB на картинку) перед добавлением в репозиторий.
